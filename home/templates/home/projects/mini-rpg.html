Project: MiniRPG

Description

MiniRPG is a game written in the GodotEngine framework. It's pretty early at this point, but the aim is a mix of old-school RPG's on the NES/SNES and more modern RPG/MMORPGs.

Game - Note: This currently requires a desktop computer to play, as we use the keyboard and mouse to control the game.
