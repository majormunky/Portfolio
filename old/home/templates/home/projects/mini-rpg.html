Project: MiniRPG

Description

MiniRPG is a game written in the GodotEngine framework. It's pretty early at this point, but the aim is a mix of old-school RPG's on the NES/SNES and more modern RPG/MMORPGs.

Features

  • Saving and Loading Games
  • NPC Dialog
  • Changing Maps
  • Multiple Characters
Note: This game isn't really much of a game right now, its more a prototype to figure out how to implement different game mechanics.
